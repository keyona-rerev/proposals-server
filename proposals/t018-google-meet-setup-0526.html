Before we start

What You Need

Checklist
  • A Google Workspace account — tied to a work domain. Gemini meeting notes are a Workspace feature.
  • A Google Drive folder where your Meet transcripts will land — you'll grab the folder ID from the URL.
  • An Anthropic API key — get one at console.anthropic.com. Free to sign up, pay-as-you-go.
  • A Super Connector API key — from your SC setup. Needed for contact syncing. (Skip if not using SC.)
  • At least one meeting notes doc already in the folder to test against.
🖥️ During this session

You'll be sharing your screen and doing the clicking. Walk through each step in order. The Files section below has all the code — everything is copy-paste ready.

Step 1

1Get Your Drive Folder ID

The script monitors a specific Drive folder for new transcript docs. You need its ID before configuring the script.

  • Open Google Drive and go to the folder where your Meet notes will land (or create one)

  • Look at the URL: drive.google.com/drive/folders/YOUR_FOLDER_ID

  • Copy the long alphanumeric string after /folders/ — that's your Folder ID

  • Keep it handy — you'll paste it into Config.gs in Step 4

Configure Meet to save notes here

In Google Meet → Settings, enable Gemini meeting notes and point the save location to this Drive folder. After each call, a new Google Doc will appear there within a few minutes.

Step 2

2Create the Apps Script Project

  • In a new tab, go to script.google.com

  • Confirm you're signed into the same Google account as your Drive folder — check the avatar top right

  • Click New Project in the top left — name it Post-Meeting Engine

  • You'll see a default Code.gs file — you'll be replacing this with the 7 files below

File naming matters

Click the + icon next to "Files" to add each new file. Select Script for .gs files. For Dashboard, select HTML. Names are case-sensitive — the files call each other by name.

Step 3

3Add Script Properties

Your API keys live in Script Properties — never in the code.

  • Click the gear icon ⚙ in the left sidebar → Project Settings

  • Scroll to Script Properties → click Add Script Property

  • Add both rows below, then click Save Script Properties

Property NameWhat to EnterWhere to get it

What about folder ID and email?

Those go directly in Config.gs as hardcoded values — not in Script Properties. You'll edit them in Step 4.

Step 4

4Paste the Files & Customize

Copy each file below. Two files need customization — they're marked. Everything else is paste-and-go.

Order matters

Start with Config.gs — paste, then fill in your folder ID, email, and task list IDs. Then paste Transcriptprocessor.gs and replace the highlighted system prompt block. All other files paste as-is.

Step 5

5Enable the Tasks Advanced Service

  • In the Apps Script editor, click the + icon next to "Services" in the left sidebar

  • Find Tasks API and click Add — leave the identifier as Tasks

🖥️ What you should see

"Tasks" appears under Services. Without this, the script errors on every task creation attempt.

Step 6

6Authorize, Test & Install Trigger

  • In the function dropdown, select runEngine — click Run ▶

  • Click Review Permissions → select your account → Advanced → "Go to [project] (unsafe)" → Allow

  • Run runEngine again — check the Execution Log at the bottom

  • Open Gmail Drafts — a follow-up draft should appear if there was a transcript doc in your folder

About the "unsafe" warning

Standard for personal scripts. You're authorizing access to your own Gmail, Drive, Calendar, and Tasks — nothing external except the Anthropic API.

const ProcessingLog = {
  getProcessedIds() {
    try { const raw = PropertiesService.getScriptProperties().getProperty('PROCESSED_IDS'); return new Set(raw ? JSON.parse(raw) : []); }
    catch(e) { return new Set(); }
  },
  _saveProcessedIds(idSet) { PropertiesService.getScriptProperties().setProperty('PROCESSED_IDS', JSON.stringify([...idSet])); },
  getLog() {
    try { const raw = PropertiesService.getScriptProperties().getProperty('PROCESSING_LOG'); return raw ? JSON.parse(raw) : []; }
    catch(e) { return []; }
  },
  _appendLog(entry) {
    try {
      const log = this.getLog(); log.unshift(entry);
      if (log.length > 100) log.length = 100;
      PropertiesService.getScriptProperties().setProperty('PROCESSING_LOG', JSON.stringify(log));
    } catch(e) {}
  },
  markProcessed(fileId, fileName, attendees, todosCount, contactsUpdated, draftId, transcriptUrl) {
    const ids = this.getProcessedIds(); ids.add(fileId); this._saveProcessedIds(ids);
    this._appendLog({ 'File ID': fileId, 'File Name': fileName, 'Processed At': new Date().toISOString(),
      'Attendees': attendees, 'Todos Count': String(todosCount), 'Contacts Updated': String(contactsUpdated),
      'Draft ID': draftId || '', 'Transcript URL': transcriptUrl || '' });
  },
  markSkipped(fileId, fileName, reason) {
    const ids = this.getProcessedIds(); ids.add(fileId); this._saveProcessedIds(ids);
    this._appendLog({ 'File ID': fileId, 'File Name': fileName, 'Processed At': new Date().toISOString(),
      'Attendees': 'SKIPPED: ' + reason, 'Todos Count': '0', 'Contacts Updated': '0', 'Draft ID': '', 'Transcript URL': '' });
  }
};
Transcriptprocessor.gs Core engine — reads docs, calls Claude, dispatches all outputs
✏ Customize prompt
▼
What to change — 3 spots

1. MY_EMAIL in _getAttendeesFromCalendar() — your email, used to exclude yourself from the attendee list.

2. baseContext in _callClaude() — the "About" block. Replace the entire highlighted block using the Claude Customization Prompt below.

3. The two YOUR FIRST NAME instances in the emailStyle lines above baseContext.

Do not touch: run(), _processFile(), _getAttendeesFromCalendar() logic, _extractText(), _isGeminiNotes().

const TranscriptProcessor = {

  run() {
    const processedIds = ProcessingLog.getProcessedIds();
    const folder = DriveApp.getFolderById(CONFIG.TRANSCRIPT_FOLDER_ID);
    const files = folder.getFiles();
    const unprocessed = [];
    while (files.hasNext()) {
      const file = files.next();
      if (file.getMimeType() !== MimeType.GOOGLE_DOCS) continue;
      if (processedIds.has(file.getId())) continue;
      unprocessed.push(file);
    }
    if (!unprocessed.length) { Logger.log('No unprocessed transcripts.'); return 'No unprocessed transcripts.'; }
    unprocessed.sort((a, b) => b.getDateCreated() - a.getDateCreated());
    try { this._processFile(unprocessed[0]); return 'Processed: ' + unprocessed[0].getName(); }
    catch(e) { ProcessingLog.markSkipped(unprocessed[0].getId(), unprocessed[0].getName(), e.message); return 'Error: ' + e.message; }
  },

  _isGeminiNotes(fileName, text) {
    return fileName.toLowerCase().includes('notes by gemini') || text.includes('Suggested next steps');
  },

  _processFile(file, opts) {
    const o = Object.assign({ draft: true, tasks: true, sc_contacts: true, sc_followups: true }, opts || {});
    const text = this._extractText(file);
    if (!text) { ProcessingLog.markSkipped(file.getId(), file.getName(), 'Could not extract text'); return; }
    const isGemini = this._isGeminiNotes(file.getName(), text);
    let attendees = this._getAttendeesFromCalendar(file);
    if (!attendees.length) attendees = isGemini ? this._extractAttendeesFromGemini(text) : this._extractAttendeesFromTranscript(text);
    const transcriptUrl = 'https://docs.google.com/document/d/' + file.getId();
    const meetingDate = file.getDateCreated();
    const result = this._callClaude(file.getName(), text, attendees, isGemini, meetingDate, transcriptUrl);
    if (!result) { ProcessingLog.markSkipped(file.getId(), file.getName(), 'Claude parse failed'); return; }
    const { todos, contact_updates, follow_up_draft } = result;
    if (follow_up_draft && attendees.length) {
      const allEmails = attendees.map(a => a.email).filter(Boolean);
      if (allEmails.length) follow_up_draft.to = allEmails.join(', ');
    }
    const { todosAdded, listsUsed } = o.tasks ? OutputDispatcher.dispatchTasks(todos) : { todosAdded: [], listsUsed: {} };
    const { contactsSummary } = o.sc_contacts ? OutputDispatcher.dispatchContacts(contact_updates) : { contactsSummary: [] };
    const draftId = o.draft ? OutputDispatcher.dispatchDraft(follow_up_draft, attendees, file.getName()) : null;
    const draftLink = draftId ? 'https://mail.google.com/mail/#all/' + draftId : null;
    if (o.sc_followups) OutputDispatcher.dispatchFollowUps(contact_updates, file.getName(), meetingDate, transcriptUrl, draftLink);
    if (o.draft || o.tasks) OutputDispatcher.sendDigest({ meetingName: file.getName(), transcriptUrl, attendees, todos: todosAdded, listsUsed, contacts: contactsSummary, draftId });
    ProcessingLog.markProcessed(file.getId(), file.getName(), attendees.map(a => a.name || a.email).join(', '), todosAdded.length, contactsSummary.length, draftId, transcriptUrl);
  },

  _getAttendeesFromCalendar(file) {
    try {
      const MY_EMAIL = 'YOUR_EMAIL@DOMAIN.COM'; // ← change this
      const fileDate = file.getDateCreated();
      const windowStart = new Date(fileDate.getTime() - 24*60*60*1000);
      const windowEnd = new Date(fileDate.getTime() + 24*60*60*1000);
      const rawName = file.getName().replace(/notes by gemini( for)?/i,'').replace(/[^a-z0-9 ]/gi,' ').trim().toLowerCase();
      const searchWords = rawName.split(/\s+/).filter(w => w.length > 2);
      const events = CalendarApp.getDefaultCalendar().getEvents(windowStart, windowEnd);
      let bestEvent = null, bestScore = 0;
      events.forEach(event => { const score = searchWords.filter(w => event.getTitle().toLowerCase().includes(w)).length; if (score > bestScore) { bestScore = score; bestEvent = event; } });
      if (!bestEvent || bestScore === 0) return [];
      const token = ScriptApp.getOAuthToken();
      const res = UrlFetchApp.fetch('https://www.googleapis.com/calendar/v3/calendars/' + encodeURIComponent(CalendarApp.getDefaultCalendar().getId()) + '/events/' + encodeURIComponent(bestEvent.getId().split('@')[0]), { headers: { Authorization: 'Bearer ' + token }, muteHttpExceptions: true });
      if (res.getResponseCode() === 200) {
        const attendees = [], seen = new Set();
        (JSON.parse(res.getContentText()).attendees || []).forEach(a => { const email = (a.email||'').toLowerCase().trim(); if (!email || email === MY_EMAIL || seen.has(email) || a.resource) return; seen.add(email); attendees.push({ name: a.displayName||'', email }); });
        return attendees;
      }
      return [];
    } catch(e) { return []; }
  },

  _extractText(file) {
    try { const body = DocumentApp.openById(file.getId()).getBody().getText(); if (body && body.trim().length > 50) return body.length > 15000 ? body.substring(0,15000)+'\n\n[truncated]' : body; } catch(e) {}
    try { const res = UrlFetchApp.fetch('https://docs.google.com/feeds/download/documents/export/Export?id='+file.getId()+'&exportFormat=txt', { headers: { Authorization: 'Bearer '+ScriptApp.getOAuthToken() }, muteHttpExceptions: true }); if (res.getResponseCode()===200) { const text = res.getContentText(); if (text && text.trim().length > 50) return text.length > 15000 ? text.substring(0,15000)+'\n\n[truncated]' : text; } } catch(e) {}
    return null;
  },

  _extractAttendeesFromTranscript(text) {
    const attendees = [];
    text.split('\n').slice(0,40).forEach(line => { const e = line.match(/[\w.+-]+@[\w-]+\.[a-z]{2,}/i); const n = line.match(/^([A-Z][a-z]+(?: [A-Z][a-z]+)+)/); if (e) { const entry = { email: e[0].toLowerCase() }; if (n) entry.name = n[1]; attendees.push(entry); } });
    return attendees;
  },

  _extractAttendeesFromGemini(text) {
    const attendees = [], seen = new Set();
    text.split('\n').slice(0,20).forEach(line => { if (line.startsWith('Invited')) { line.replace('Invited','').trim().split(/[,;]/).forEach(n => { const name = n.trim(); if (name && !seen.has(name)) { seen.add(name); attendees.push({ name, email: '' }); } }); } });
    return attendees;
  },

  _callClaude(meetingName, text, attendees, isGemini, meetingDate, transcriptUrl) {
    const attendeeList = attendees.length ? attendees.map(a => a.name+(a.email?' ('+a.email+')':'')).join(', ') : 'Unknown attendees';
    const now = new Date();
    const meetingDateStr = meetingDate.toLocaleDateString('en-US',{weekday:'long',year:'numeric',month:'long',day:'numeric'});
    const sendDateStr = now.toLocaleDateString('en-US',{weekday:'long',year:'numeric',month:'long',day:'numeric'});
    const sameDay = meetingDate.toDateString() === now.toDateString();
    const daysDiff = Math.floor((now-meetingDate)/(1000*60*60*24));
    const isGroup = attendees.filter(a => a.email).length > 1;
    const timeContext = 'TIME CONTEXT:\n- Meeting date: '+meetingDateStr+'\n- Email send date: '+sendDateStr+'\n- Time since meeting: '+(sameDay?'same day':daysDiff+' day(s) ago')+'\n\nTIME RULES:\n1. NEVER use relative time words\n2. Use actual date or neutral opener\n3. Specific future deadlines ARE fine\n4. When in doubt, omit the time reference';
    const emailStyle = isGroup
      ? 'EMAIL STYLE: group reply-all. Address everyone by name. Warm, direct, specific. No generic openers. No emojis. Plain text. Close with "Warmly," then "YOUR FIRST NAME". Include near end: "Meeting notes: '+transcriptUrl+'"'
      : 'EMAIL STYLE: warm, direct, specific. No generic openers. No emojis. Plain text. Lead with something concrete. Close with "Warmly," then "YOUR FIRST NAME". Include near end: "Meeting notes: '+transcriptUrl+'"';

    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
    // CUSTOMIZE THIS BLOCK — use the Claude Prompt below to generate
    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
    const baseContext =
      'You are a post-meeting intelligence assistant for [YOUR NAME].\n\n'+
      'ABOUT [YOUR NAME]:\n'+
      '- [Venture 1]: [one-line description]\n'+
      '- [Venture 2]: [one-line description]\n\n'+
      'CONTACT INTELLIGENCE: For each person identify role: Funder/Customer/Partner/Distribution/Talent\n'+
      'TASK ROUTING ventures: [keywords matching your TASK_LISTS keys in Config.gs]\n\n'
      + emailStyle+'\n\n'+timeContext
      +'\n\nReturn ONLY raw JSON. No markdown fences. No preamble.\nSTRUCTURE:\n{"todos":[{"task":"string","notes":"string","venture":"string"}],"contact_updates":[{"email":"string","full_name":"string","title_role":"string","organization":"string","notes":"string","what_building":"string","what_need":"string","what_offer":"string","venture_relevance":"string","next_action":"string","next_action_date":"string"}],"follow_up_draft":{"to":"string","subject":"string","body":"string"}}';
    // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    const instructions = isGemini ? '\n\nGEMINI NOTES: Extract todos from Suggested next steps. Extract all participant names. Draft warm follow-up.' : '\n\nRAW TRANSCRIPT: Extract action items, attendee intel, draft follow-up.';
    const payload = { model: 'claude-sonnet-4-20250514', max_tokens: 2000, system: baseContext+instructions, messages: [{ role: 'user', content: 'Meeting: '+meetingName+'\nAttendees: '+attendeeList+'\n\n'+text }] };
    const response = UrlFetchApp.fetch('https://api.anthropic.com/v1/messages', { method: 'post', contentType: 'application/json', headers: { 'x-api-key': CONFIG.ANTHROPIC_API_KEY, 'anthropic-version': '2023-06-01' }, payload: JSON.stringify(payload), muteHttpExceptions: true });
    const data = JSON.parse(response.getContentText());
    if (!data.content || !data.content[0]) { Logger.log('Claude error: '+response.getContentText()); return null; }
    const raw = data.content[0].text.trim();
    try { return JSON.parse(raw); } catch(e) { try { return JSON.parse(raw.replace(/^```json\n?/,'').replace(/```$/,'').trim()); } catch(e2) { return null; } }
  }
};

If something goes wrong

Troubleshooting

No draft appeared after running manually
Check the Execution Log for errors. Confirm there's an unprocessed Google Doc in your transcript folder (not a PDF — must be a Google Doc). Verify TRANSCRIPT_FOLDER_ID in Config.gs.
Script runs but creates no tasks
Confirm the Tasks Advanced Service is enabled (Step 5). Verify your task list IDs in Config.gs — an incorrect ID fails silently.
Claude parse failed in the Execution Log
Check the baseContext string for unescaped quotes or broken concatenation. Run pingClaude() first to confirm the API key returns a 200 status.
Attendees are wrong or missing
The script matches the doc filename to a Calendar event title. If no match, it falls back to extracting emails from the first 40 lines. Make sure your calendar event title shares words with the meeting notes filename.
Authorization error after trigger fires
Run runEngine() manually to re-authorize, then delete and re-add the trigger.
I want to reprocess a specific file
Open the dashboard (Web App URL). Click the entry in the Processing Log, toggle which outputs you want, click Reprocess.
